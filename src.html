Cách tải file code về máy:

Cách 1: Qua GitHub (Khuyên dùng)

  1. Kết nối GitHub (nút GitHub góc trên phải)
  2. Create Repository
  3. Mở Terminal trên máy và chạy:
git clone https://github.com/your-username/your-repo-name.git

Cách 2: Download trực tiếp

  1. Sau khi kết nối GitHub, vào repository trên GitHub
  2. Nhấn nút Code màu xanh
  3. Chọn Download ZIP
  4. Giải nén file ZIP về máy

Cách 3: Copy thủ công (không khuyến khích)

Vì bạn đang trong Code Editor:

  1. Chọn từng file bên trái
  2. Copy code
  3. Paste vào editor trên máy (VS Code, Notepad++...)

Khuyến nghị: Dùng Cách 1 hoặc 2 để có đầy đủ cấu trúc project và dễ quản lý code!
